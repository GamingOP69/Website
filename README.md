# GamingOP — Website

Production-ready Next.js + Tailwind website for the GamingOP creator hub. Features:

- Server-side YouTube integration (latest, popular, live, trending)
- Minecraft server status and join instructions
- Discord embed, events, contact form
- SEO helpers, sitemap generation, security middleware
- Vercel deployment-ready with CI

See `README_SCAFFOLD.md` for setup and deploy instructions.
